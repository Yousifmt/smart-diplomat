# **App Name**: Smart Diplomat

## Core Features:

- User Authentication: Secure email/password sign up, sign in, and password reset using Firebase Authentication.
- Country & Role Selection: Onboarding process to select role (Envoy/Analyst/Admin), default country/region, and language preference.
- AI-Powered Chat: Chat interface enabling users to ask questions and receive answers from an AI assistant grounded in approved sources.
- Retrieval Pipeline: Fetch contextually relevant information, from the approved source list. Refine queries based on approved domains.
- Intelligent Referencing tool: AI-powered functionality providing a References section with numbered citations linked to the source URLs used to create a response.
- Admin Dashboard: Admin interface for managing allowed sources, viewing usage analytics, and safety flags.
- Firestore Integration: Utilize Cloud Firestore for storing user data, chat logs, messages, source catalog, audit logs, and optional usage counters.

## Style Guidelines:

- Primary color: Navy blue (#00308F) evoking trust and competence.
- Background color: Light gray (#F0F0F0), a clean, neutral background.
- Accent color: Gold (#FFD700) which is analogous to navy blue, but set apart in both brightness and saturation.
- Headline Font: 'Playfair' (serif) with a high-end feel; Body Font: 'PT Sans' (sans-serif) which is modern, and complements Playfair
- Use professional, minimalist icons for navigation and actions.
- Responsive layout for optimal viewing on different devices.
- Subtle transitions and loading animations to enhance user experience.
